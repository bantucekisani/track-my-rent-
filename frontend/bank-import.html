Matching Rules

How matching works

Track My Rent checks transaction details before posting anything to the ledger.

  • Matches tenant by reference, phone, email, or name.
  • Verifies active lease before posting.
  • Auto-posts high-confidence matches to the ledger.
  • Flags uncertain matches for manual approval.
  • Prevents duplicate payments.
